Aplikasi ini adalah proyek Android sederhana berbasis Kotlin yang menampilkan antarmuka pengguna (UI) untuk menyapa pengguna. Aplikasi ini mendemonstrasikan penggunaan komponen dasar Android seperti EditText, Button, TextView, dan Switch.

&nbsp;

**Cara Menjalankan**

1\. Buka folder proyek ini menggunakan **Android Studio**.

2\. Tunggu proses **Gradle Sync** selesai.

3\. Klik tombol **Run** pada toolbar untuk menjalankan aplikasi di emulator atau perangkat fisik.
